# prettier-tasks

[![styled with prettier](https://img.shields.io/badge/styled_with-prettier-ff69b4.svg)](https://github.com/prettier/prettier)

## Install

```bash
npm install
```

## Test Lint

```
npm run lint
```

## Tasks

```
npm run lint:fix
npm run lint:script
npm run lint:style
npm run lint:json
```
